# Suggestions d'Améliorations PowerAlert

Ce document détaille les pistes d'amélioration envisagées pour augmenter la proposition de valeur de l'application PowerAlert.

## Santé de la batterie (Battery Health)
- **Description** : Affichage d'un dashboard complet sur l'état physique du matériel.
- **Fonctionnalités réalisées** :
    - Dashboard adaptatif masquant automatiquement les données indisponibles (Cycles).
    - Calcul de l'usure réelle et affichage des capacités (Design vs Actuel).
    - Monitoring thermique et de puissance (Watts) avec tooltips pédagogiques.

## Historique et Analytics
- **Description** : Permettre à l'utilisateur de suivre l'évolution de son autonomie sur le long terme.
- **Fonctionnalités** :
    - Graphiques montrant l'évolution de l'autonomie au fil des mois.
    - Analyse des tendances de consommation.

## Smart Automation (Notifications Intelligentes)
- **Description** : Étendre la portée des alertes au-delà du poste de travail.
- **Fonctionnalités** :
    - Intégration de notifications smartphone via une API externe.
    - Prévenir l'utilisateur de l'état de sa batterie lorsqu'il n'est pas devant son PC.
