# Release v1.0.0 - Excellence du Suivi de Batterie

### Informations générales
* Tag : v1.0.0
* Date : 31 Mars 2026
* Statut : Stable / Finalisé

### Description
Cette première version majeure de PowerAlert marque l'aboutissement du développement du système de surveillance de la batterie et de l'interface utilisateur. L'application offre désormais un suivi précis, une intégration parfaite dans la barre des tâches Windows et un système de notifications intelligent conçu pour ne jamais être intrusif tout en restant informatif.

### Nouveautés
* Surveillance différentielle haute fréquence : Implantation d'un moteur de détection pour les changements d'état de charge et les niveaux critiques.
* Menu Tray personnalisé : Menu contextuel moderne basé sur la technologie web pour un accès rapide aux informations de la batterie.
* Notifications intelligentes : Système de notifications natif stabilisé avec gestion du comportement en arrière-plan.
* Lancement minimisé : Option permettant de démarrer l'application directement dans la barre d'état.
* Installateur automatisé : Configuration complète pour la génération automatique d'un installateur Windows sécurisé.

### Améliorations
* Intégration graphique native : Migration vers des ressources SVG locales pour une meilleure netteté visuelle et une indépendance vis-à-vis des serveurs tiers.
* Optimisation de la production : Réduction drastique des logs en mode production pour améliorer les performances et la stabilité.
* Sobriété documentaire : Refonte du README et de la documentation technique pour une clarté professionnelle accrue.

### Corrections
* Stabilisation des notifications : Résolution d'un problème de notifications en boucle lors de changements d'état rapides.
* Robustesse de l'affichage : Gestion des situations où les données de la batterie sont momentanément indisponibles, évitant ainsi les erreurs graphiques.
* Cohérence du tray : Correction de l'affichage des informations de charge dans le menu réduit.
